Realtime BINGO Game

A real-time online bingo game platform where multiple players can join different virtual rooms to play bingo together. Players receive digital 3 bingo cards with random numbers, and as numbers are drawn in real-time, their cards are manually marked so that it is interactive. The system uses WebSocket connections (socket.io) to ensure instant updates for all players, showing number draws and win conditions. Players can chat with each other while playing, and the system tracks wins. The platform includes security measures to ensure fair play and can handle multiple concurrent games across different rooms.
